# osg & osgearth learning

osg & osgearth 的一些示例。

## 环境

- CMake 3.21.1
- Visual Studio 2019
- VSCode
- osg3.6.5
- vcpkg

## 安装编译osg

- `.\vcpkg.exe install osg:x64-windows`
- `.\vcpkg.exe export osg:x64-windows --7zip`

[编译好的osg3.6.5库](https://pan.baidu.com/s/1fCgsFzuM5w0jAaY-iMLT9w?pwd=s985)
